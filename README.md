# TECH-TES-MII

## Instruksi Menjalankan Program


### Masuk ke folder proyek
```bash
cd PROJECT_FOLDER
```

### Jalankan Docker Compose
```bash
docker-compose up
```

### Instal dependensi
```bash
npm install
```

### Migrasi basis data
```bash
npx sequelize-cli db:migrate
```

### Jalankan seed
```bash
npx sequelize-cli db:seed:all
```

### Jalankan program
```bash
npm run start:dev
```
