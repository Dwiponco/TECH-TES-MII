# TECH-TES-MII

## Instruksi Menjalankan Program


### Masuk ke folder proyek
```bash
cd PROJECT_FOLDER
```

### Jalankan Docker Compose DB
```bash
docker-compose -f docker-compose-db.yml up
```

### Jalankan Docker Compose APP
```bash
docker-compose -f docker-compose-app.yml up
```
